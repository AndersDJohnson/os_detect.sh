# os_detect.sh
OS detection for Bash.

Normalizes current OS down to a few limited types. See [os_detect.sh](os_detect.sh) for those values.

## Usage

As a script:

```sh
./os_detect.sh/os_detect.sh
```

As a function:

```sh
source os_detect.sh/os_detect.sh

OS=os_detect
```
